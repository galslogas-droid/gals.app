# GALS NextGen v6.3

Aplikasi manajemen pangkalan gas dan galon.

## Fitur
- ✅ Dashboard penjualan realtime
- ✅ 8 kartu analisis (Omzet, Kulakan, Operasional, Jajan, Zakat, Shodaqoh, Laba, Setoran)
- ✅ Manajemen stok gas
- ✅ Rekap transaksi dengan search
- ✅ Kas admin (jatah vs real)
- ✅ Grafik omzet & laba
- ✅ Analisis bisnis + score
- ✅ Dark mode
- ✅ Auto-save
- ✅ Responsive untuk HP

## Teknologi
- HTML5, CSS3, JavaScript
- Google Apps Script (backend)
- Google Sheet (database)
- Chart.js (grafik)

## Cara Pakai
1. Buka link: https://[nama-app].netlify.app
2. Input penjualan di form
3. Lihat dashboard otomatis update
